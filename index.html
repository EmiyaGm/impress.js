Bootstrap是什么

  • Bootstrap 是一个用于快速开发 Web 应用程序和网站的前端框架
  • Bootstrap 是基于 HTML、CSS、JAVASCRIPT 的
  • Bootstrap 是由 Twitter 的 Mark Otto 和 Jacob Thornton 开发的
  • Bootstrap 是 2011 年八月在 GitHub 上发布的开源产品

Bootstrap做什么

  • 提供样式和组件快速写网站
  • 一个框架、多种设备 -----响应式
  • 一个框架、多种浏览器 -----兼容性

栅格系统

栅格系统用于通过一系列的行（row）与列（column）的组合来创建页面布局，你的内容就可以放入这些创建好的布局中。

随着屏幕或视口（viewport）尺寸的增加，系统会自动分为最多12列。

到目前为止，Bootstrap已经发展了四个版本，目前使用较广的是版本2和3，其中2的最新版本的是2.3.2，3的最新版本是3.3.7。
在2017年8月下旬，Bootstrap四周岁之际，Bootstrap团队发布了Bootstrap 4 alpha版
截止到目前，Bootstrap 4 已经更新至4.1.3

关于v2 v3 v4三个版本的区别综述

Bootstrap 2 和 Bootstrap 3


v2和v3之间的差别主要体现在增加了一些新的特性，对于一些类也进行了调整，两个版本在使用的方法上是没什么大的区别的。
例：在bootstrap2.x的栅格类标签使用的是.span*，但在新版本中这一标签不再保留，而是使用了.col-md-*替代

值得关注的变化主要是:
1、bootstrap.css bootstrap-response.css在v3中整合到bootstrap.css中，所以在 Bootstrap 2 中，对框架中的某些关键部分增加了对移动设备友好的样式。而在 Bootstrap 3 中，重写了整个框架，使其一开始就是对移动设备友好的。Bootstrap 是移动设备优先的。
2、Bootstrap3对于IE浏览器的兼容，从IE8开始，如果你需要对更低版本的IE浏览器兼容，请选择 Bootstrap 2。
3、组件样式设计风格的变化，如果你是喜欢渐变凹凸风格的，就可以选择采用Bootstrap2来进行开发；反之，如果你喜欢扁平化设计风格，那么就可以采用Bootstrap3了。

Bootstrap 3 和 Bootstrap 4


v3和v4之间的差别就比较大了，除了基本的类的调整外，整个Bootstrap4的底层css使用sass进行了重写，从原来的Less迁移到Sass，这就使得编译打包之后的bootstrap.css比之前文件大小减少了40%，大大加快了加载速度。
并且栅格系统等组件只使用 flexbox 来构建而非之前那样根据 $enable-flex 变量而定，这就导致了一个重大的浏览器兼容上的问题，Bootstrap4的布局将不兼容IE9及其以下版本，同时也不再支持IOS 6。
并且为了更好的适配移动端，更好的提供响应式布局，使用rem和em代替px单位。

兼容性详谈


Bootstrap2中， 对于低版本浏览器的兼容都是不错的，特别是当你需要兼容IE8以下版本浏览器的时候。但是要注意一点，部分JavaScript的组件，例如进度条，当中如果依赖于CSS3的渐变、过度和动画特性的时候，这些都是不被IE7-9 或较老版本的Firefox所支持的。并且IE10和Opera 12之前的版本均不支持动画特性

响应式详谈


在v2中，我们需要多引入一个css文件，bootstrap-responsive.css，这点跟v3版本不同，v3将其和bootstrap.css文件打包在了一起，这样Bootstrap 会自动帮你针对不同的屏幕尺寸调整你的页面，使其在各个尺寸的屏幕上表现良好。

如果你不想使用响应式布局，又不想回退至v2版本，官方也提供了禁用这一特性的方法。

从v2开始确立的bootstrap的响应式设计，使用的是 Media Queries ，也就是我们所说的媒体查询。在bootstrap2中，常用的媒体尺寸主要有五种。

而在v3中，bootstrap增强了对响应式的支持，其中一个重要的要点就是移动设备优先，所以他将响应式布局和栅格系统结合在一起，利用四种类前缀。

并且针对性的提供了响应式工具类，使用.visible和.hidden针对不同屏幕尺寸隐藏或显示页面内容。

从 v3.2.0 版本起，形如 .visible-*-* 的类针对每种屏幕大小都有了三种变体，每个针对 CSS 中不同的 display 属性。

值得注意的是在v4中分为了五类。

v3升级到v4的时候需要注意，大量v3中的响应式工具类被修改，增加了特大屏xl尺寸名词，并且所有尺寸名词都对应调小了一级。
